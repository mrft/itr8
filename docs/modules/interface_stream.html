    Really useful combined with the lineByLine() operator to process large files.

    (If you want the raw bytes, simply use itr8FromStream(process.stdin))

itr8FromStream

  • itr8FromStream(stream: Readable): TPipeable<any, any> & AsyncIterableIterator<any>
    • Defined in interface/stream.ts:19

    Transforms a stream into an async itr8.

    To open a file as an iterator:

    itr8FromStream(fs.createReadStream("D://data.txt"));

    How it works: the stream.on data handler will put everything in a buffer and pause the stream until next has been called on the iterator, which will cause the stream to resume

    Parameters

    • stream: Readable

    Returns TPipeable<any, any> & AsyncIterableIterator<any>
